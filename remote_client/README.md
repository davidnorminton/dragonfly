# Remote Client for Dragonfly

This directory contains tools and documentation for connecting to the Dragonfly Home Assistant from remote devices.

## Files

- `remote_client.py` - Python script for sending data from remote devices
- `REMOTE_CONNECTION_GUIDE.md` - Complete guide for remote connections
- `README.md` - This file

## Quick Start

1. Copy `remote_client.py` to your remote device
2. Install dependencies: `pip3 install websockets`
3. Run: `python3 remote_client.py <SERVER_IP> --interactive`

See `REMOTE_CONNECTION_GUIDE.md` for detailed instructions.
